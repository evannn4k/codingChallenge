Faktorial

cara kerja function penghitung faktorial

1. Menerima parameter number
2. Cek apakah negative atau bukan
3. Loop number jika number lebih dari 0 lalu di decrement
4. Number yang di loop selalu dikalikan dengan result sampai numbernya = 1

cara kerja function pembuat diamond star

1. Menerima parameter number
2. Cek apakah negative atau bukan
3. Perulangan dibagi menjadi dua, yaitu perulangan atas dan perulangan bawah

Perulangan atas

1. Membuat skip column dengan menghitung number yang di decrement
2. Membuat bintang dengan mengalikan mengalikan number dengan angka 2 dan di increment

Perulangan bawah

1. Sama seperti perulangan atas namun ada logika yang saya balik disini
